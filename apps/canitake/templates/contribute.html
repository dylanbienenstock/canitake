Thank you

...for showing an interest in contributing to this project. We pull data from two sources. Tripsit.me, which is managed by a lovely group of people, contains information about (primarily) illicit substances, as well as nootropics and various supplements. Our second source, RxNorm, is managed by the U.S. National Library of Medicine, and contains information about prescription and OTC medications. The goal of this project is to create a database that bridges the gap between those two worlds, in the name of harm reduction.
